# 記錄本次對話重點到 SCRATCHPAD.md

依序執行：

1. **取得當前時間**，格式為 `YYYY-MM-DD HH:mm`。
2. **摘要本次對話**，提取以下資訊：
   - 做了什麼（一句話）
   - 碰到的問題或障礙（如果有）
   - 做了哪些決策或取捨（如果有）
3. **寫入 `SCRATCHPAD.md`**，附加在檔案最後，格式如下：

    ```markdown
    ## YYYY-MM-DD HH:mm

    **做了什麼**
    - ...

    **問題 / 障礙**
    - ...（無則省略）

    **決策 / 取捨**
    - ...（無則省略）

    ---
    ```

4. 確認寫入完成，回報「✅ 已記錄到 SCRATCHPAD.md」。

**注意：只附加（append），不覆蓋既有內容。**
